Project Overview

VitScan AI is a healthcare innovation designed to solve the global problem of hidden hunger. Many vitamin deficiencies show physical symptoms on the eyes, tongue, and skin before they show up in blood tests.

Our project uses **Flask** and **Deep Learning** to create a seamless bridge between a user and a diagnosis. The Admin trains the system with thousands of medical images, and the user gets a result in under 3 seconds.
